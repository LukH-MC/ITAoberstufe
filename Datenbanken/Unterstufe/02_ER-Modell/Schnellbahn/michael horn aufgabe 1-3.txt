Relationen
1:
	Ein Zug hat zwei, oder mehrere Wagen.
	Ein Wagen gehört zu genau einem Zug.
2:
	Ein Zug hat keine, eine, oder mehrere Linien.
	Eine Linie hat einen, oder mehrere Züge.
3:
	Eine Linie hat eine, oder mehrere Fahrten.
	Eine Fahrt wird von genau einer Linie gefahren.
4:
	Eine Fahrt hat zwei, oder mehrere Bahnhöfe.
	Ein Bahnhof hat keine, eine, oder mehrere Fahrten
5:
	Ein Zug wird auf genau einem Bahnhof abgestellt.
	Ein Bahnhof hat keinen, einen, oder mehrere Züge auf seinem Abstellgleis.
6:
	Eine Fahrt wird von genau einem Zug gefahren.
	Ein Zug kann keine, eine, oder mehrere Fahrten fahren.
7:
	Eine Linie hat genau zwei Bahnhöfe als Grenzpunkt.
	Ein Bahnhof ist Grenzpunkt von keiner, einer, oder mehreren Linien.
8:
	Ein Fahrer fährt keine, eine, oder mehrere Fahrten.
	Eine Fahrt wird von genau einem Fahrer gefahren.
